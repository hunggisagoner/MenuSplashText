# Menu Splash Text

**Menu Splash Text** is a [Geode](https://geode-sdk.org) mod that adds random splash texts to the main menu of Geometry Dash.

## Features
- Displays a random splash text on the main menu
- Supports custom splash texts from a file
- Simple and lightweight

## Custom Splash Text

You can load your own splash texts using a text file.

Example format:

hello world;this is a splash text;another splash;geometry dash

Use a semicolon (;) to separate splash texts.

## Settings

- **Use Custom Splash Text**  
  Enable loading splash texts from a custom file.

- **Custom Splash File**  
  Select a text file containing your splash texts.
