# Menu Splash Text
Adds random splash text to the Geometry Dash main menu.

# Customizable
* This mod lets you customize the splash text shown on the main menu, replacing the default one with your own messages.  
* You can add multiple splash texts, and one will be randomly displayed each time the menu loads.
* You must use a semicolon (;) to separate splash texts.
